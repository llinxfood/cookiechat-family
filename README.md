# cookiechat-family

Scaffold de `CookieChat` para iPad con chat familiar cerrado (`family_main`).

## Contenido

- Codigo SwiftUI base listo para copiar a un proyecto Xcode.
- Repositorio mock para arrancar sin backend.
- Repositorio Firestore opcional para produccion.
- Pantalla de chat y panel de padres (solo admin).
